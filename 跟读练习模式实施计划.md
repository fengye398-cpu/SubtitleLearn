# 跟读练习模式实施计划

## 项目信息
- **项目名称**: 视频字幕导出工具 - 跟读练习模式集成
- **创建日期**: 2025-12-14
- **目标**: 将Aegisub脚本中的跟读练习功能集成到GUI应用中
- **风险等级**: 中等（新增功能，不修改现有模式）

---

## 一、项目概述

### 1.1 功能描述
在现有的视频字幕导出GUI应用中新增"跟读练习模式"，实现：
- 每个句子重复播放N次（1-10次可配置）
- 句子间自动添加停顿时间（支持按倍数或固定秒数）
- 停顿时可选择是否显示字幕
- 生成适合语言学习的跟读练习视频

### 1.2 参考实现
- **参考文档**: `D:\Program Files\Aegisub\automation\autoload\ffmpeg_cut_complete.lua`
- **目标应用**: `D:\桌面\video_subtitle_cut\video_subtitle_app`
- **核心文件**: `ui/dialogs/integrated_export_dialog.py`

### 1.3 技术特点
- 采用新增模式，不修改现有"标准模式"和"重新编码"
- 复用现有的FFmpeg切割和字幕处理逻辑
- 通过JSON配置传递shadowing参数到后端

### 2.2 跟读模式参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| repeat_count | int | 3 | 播放次数（1-10） |
| pause_between_repeats | bool | true | 是否在重复间停顿 |
| pause_mode | str | "multiplier" | 停顿模式（multiplier/fixed） |
| pause_multiplier | float | 1.5 | 按倍数：句子时长×倍数 |
| pause_seconds | float | 2.5 | 固定秒数 |
| show_pause_subtitles | bool | true | 停顿时是否显示字幕 |
| crf | int | 24 | 视频质量（0-51） |
| preset | str | "veryfast" | 编码预设 |

### 2.3 处理流程

```
1. 用户选择字幕片段
2. 选择"跟读练习模式"
3. 配置跟读参数
4. 后端处理：
   ├─ 切割原始片段
   ├─ 生成静音/黑屏片段（停顿用）
   ├─ 按规则拼接：[片段1] + [停顿] + [片段1] + [停顿] + [片段1] + [间隔] + [片段2] + ...
   ├─ 生成对应的字幕文件
   └─ 合并最终视频
5. 输出跟读练习视频
```

#### 核心算法
```python
# 伪代码
for segment in segments:
    for i in range(repeat_count):
        # 添加原始片段
        add_segment(segment)

        # 添加停顿（除了最后一次重复）
        if pause_between_repeats and i < repeat_count - 1:
            pause_duration = calculate_pause(segment, pause_mode)
            add_pause(pause_duration, show_subtitles)

    # 片段间的间隔（可选）
    add_gap_between_segments()
```
