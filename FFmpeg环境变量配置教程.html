🎥 FFmpeg 简介

什么是 FFmpeg？

FFmpeg 是一个开源的音视频处理工具，可以进行：

  • ✅ 视频格式转换（MP4、AVI、MKV等）
  • ✅ 音频提取和转换
  • ✅ 视频剪辑和合并
  • ✅ 视频压缩和优化
  • ✅ 字幕处理

为什么要配置环境变量？

配置环境变量后，可以在任意目录下直接使用 ffmpeg 命令，无需输入完整路径。

📥 下载 FFmpeg

步骤 1：访问官网

打开浏览器，访问：https://ffmpeg.org/download.html

步骤 2：选择 Windows 版本

  • 点击 Windows 图标下的链接
  • 推荐选择 gyan.dev 或 BtbN 提供的版本
  • 下载 ffmpeg-release-full.7z

步骤 3：解压文件

将下载的文件解压到合适位置，例如：

C:\Program Files\ffmpeg

⚠️ 路径不要包含中文或特殊字符

⚙️ 添加环境变量（方法一）

步骤 1：打开系统属性

  • 右键点击 此电脑（或"我的电脑"）
  • 选择 属性
  • 点击左侧 高级系统设置

步骤 2：打开环境变量

  • 在"系统属性"窗口中，点击 环境变量 按钮

⚙️ 添加环境变量（方法一续）

步骤 3：编辑 Path 变量

  • 在"系统变量"区域找到 Path 变量
  • 选中后点击 编辑
  • 点击 新建
  • 输入 FFmpeg 的 bin 目录路径，例如：
C:\Program Files\ffmpeg\bin

步骤 4：保存设置

  • 点击 确定 保存所有窗口
  • ⚠️ 必须关闭所有打开的命令行窗口，新配置才会生效

⚙️ 添加环境变量（方法二：命令行）

快速配置方法（需管理员权限）

按 Win + X，选择"Windows PowerShell (管理员)"，运行：

[System.Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\Program Files\ffmpeg\bin", "Machine")

⚠️ 注意替换为你的实际安装路径

✅ 验证是否配置成功

步骤 1：打开命令提示符

  • 按 Win + R 打开运行窗口
  • 输入 cmd 并回车

步骤 2：检查版本

在命令行中输入：

ffmpeg -version

如果配置成功，会显示 FFmpeg 的版本信息 ✅

✅ 验证结果示例

✅ 成功的输出示例

ffmpeg version 6.0-full_build Copyright (c) 2000-2023 the FFmpeg developers built with gcc 12.2.0 (Rev10, Built by MSYS2 project) configuration: --enable-gpl --enable-version3 ... ...

✓ 看到类似输出说明配置成功！

❌ 失败的输出示例

'ffmpeg' 不是内部或外部命令，也不是可运行的程序或批处理文件。

✗ 出现此错误说明配置失败，请检查路径是否正确

🔍 其他验证命令

检查完整路径

where ffmpeg

显示 ffmpeg.exe 的完整路径，例如：

C:\Program Files\ffmpeg\bin\ffmpeg.exe

简单测试转换功能

ffmpeg -h

显示帮助信息，确认命令可用

❓ 常见问题解答

问题 1：配置后仍提示"不是内部或外部命令"

  • ✓ 检查路径是否正确（必须是 bin 目录）
  • ✓ 关闭所有命令行窗口后重新打开
  • ✓ 重启电脑使配置生效

问题 2：路径中包含空格

如果路径包含空格（如 C:\Program Files），在 Path 中可以：

  • ✓ 直接添加，Windows 会自动处理
  • ✓ 或使用双引号包裹路径（某些旧系统需要）

❓ 常见问题解答（续）

问题 3：修改"用户变量"还是"系统变量"？

  • 用户变量：只对当前用户有效
  • 系统变量：对所有用户有效（推荐）
  • 建议修改系统变量中的 Path

问题 4：如何卸载或更换版本？

  • 删除 FFmpeg 文件夹
  • 从环境变量中移除对应路径
  • 重新下载新版本并配置即可

📝 总结

完整配置流程

  1. 下载 FFmpeg 并解压到指定目录
  2. 找到 bin 文件夹的完整路径
  3. 将该路径添加到系统环境变量 Path 中
  4. 关闭所有命令行窗口
  5. 重新打开命令行，运行 ffmpeg -version 验证

🎉 配置成功后你可以：

  • 在任意位置使用 FFmpeg 命令
  • 运行依赖 FFmpeg 的程序和脚本
  • 进行音视频处理工作
