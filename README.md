1.SubtitleLearn 是一款专为外语学习者设计的视频字幕处理工具。通过智能的字幕搜索和片段提取功能，帮助用户高效学习外语。SubtitleLearn is a video subtitle processing tool designed specifically for foreign language learners. It leverages intelligent subtitle search and clip extraction features to help users learn foreign languages efficiently.

4.教程视频：https://www.bilibili.com/video/BV13Us2zYEyb/
